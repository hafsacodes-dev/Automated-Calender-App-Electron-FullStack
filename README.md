# Automated-Calender-App-Electron-FullStack
A fun twist on a mini desktop calender app, built using the Electron Framework



# Features
 **Native Desktop Experience**: Runs seamlessly and automates appropriately based on the weekly days, a fun little project
*   **Clean & Responsive UI**: A spongbob inspired twist on a calender app, response time is also adequate
*   **Lightweight Repository**: Built following the vanilla pack: Html-Css-Javascript, keeping the logic adaptable and easy to follow.




## Tech Stack
*   **Frontend**: HTML5, CSS3, JavaScript (ES6+)
*   **Runtime Environment**: Node.js
*   **Desktop Framework**: Electron



## Project Structure

```text
my-calender-app/
├── assets/                 # UI graphics and localized image assets
```
